# Hackathon DevConnect – Projeto FlixDev

Este repositório documenta um modelo de avaliação prática utilizado no curso de Desenvolvimento de Sistemas.

A atividade simula um ambiente corporativo real onde duas equipes trabalham em um sistema backend utilizando um fluxo de handoff entre desenvolvimento e manutenção.

O projeto foi criado para desenvolver habilidades técnicas e colaboração em ambientes de desenvolvimento.
